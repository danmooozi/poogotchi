# PooGotchi (똥가치) 제품 요구사항 정의서 (PRD)

## 1. 제품 개요

- **컨셉**: 레트로 다마고치 감성의 웹 기반 캐주얼 육성 게임. "더러움"에서 시작해 "귀여움"으로 진화하는 반전 재미를 선사합니다.
- **핵심 가치**: 단순한 루프, 수집 욕구 자극, 짧은 플레이 타임 (1~3분).

## 2. 주요 화면 구성 (Proposed Screen List)

핵심 화면 구성 디자인은 프로젝트 루트 stitch_pixel_poop_evolution 폴더를 참고하세요

1. **인트로/생성 화면 (Splash & Creation)**: 게임 시작 연출 및 첫 캐릭터 생성.
2. **메인 게임 화면 (Main Game Loop)**: 캐릭터 육성, 상태 확인, 액션 수행의 중심.
3. **진화/성장 연출 화면 (Evolution Overlay)**: 레벨업 및 진화 시 발생하는 축하 연출.
4. **컬렉션/도감 화면 (Collection/Dex)**: 지금까지 발견한 진화 형태 및 업적 확인.
5. **설정/꾸미기 팝업 (Settings & Decor)**: 배경 변경 및 악세사리 장착.

## 3. 핵심 기능 상세

- **캐릭터 시스템**: 픽셀 스타일의 똥 캐릭터. 레벨(1, 3, 5, 10, 15+)에 따라 외형 진화.
- **상태 시스템**: 배고픔, 행복도, 청결도 실시간 감소.
- **육성 액션**: 먹이 주기(🍔), 닦아주기(🧼), 놀아주기(🎾).
- **진화 조건**: 상태 수치에 따른 분기 (예: 청결도 중심 -> 깨끗한 똥, 식사 중심 -> 뚱뚱한 똥).

## 4. 디자인 가이드라인

핵심 디자인 시스템은 프로젝트 루트 DESIGN.md 파일을 참고하세요.

- **비주얼 스타일**: 8-bit/16-bit 픽셀 아트, 비비드한 컬러, 둥근 UI 요소.
- **인터랙션**: 클릭/터치 중심의 직관적 UX, 레트로 사운드 효과를 시각화한 애니메이션.

## 5. 기술 범위

### 프론트

- React, Vite
- Canvas 또는 DOM 애니메이션
- 상태 관리: Zustand / Context

### 백엔드 (선택)

- 초기: localStorage
- 확장:
  - 유저 데이터 저장
  - 랭킹 시스템
